# CRM App UMYFA

Una aplicación CRM simple y eficiente para gestionar clientes y notas, construida con tecnologías web nativas.

## 🚀 Características

- **Gestión de Clientes**: Crear, editar, visualizar y eliminar información de clientes
- **Sistema de Notas**: Agregar y gestionar notas asociadas a cada cliente
- **Interfaz Intuitiva**: Diseño limpio y fácil de usar
- **Responsive**: Funciona perfectamente en dispositivos móviles y desktop
- **Almacenamiento Local**: Los datos se guardan localmente en el navegador

## 🛠️ Tecnologías

- **HTML5**: Estructura y semántica
- **CSS3**: Estilos y diseño responsive
- **JavaScript ES6+**: Lógica de aplicación y manipulación del DOM
- **LocalStorage**: Persistencia de datos del lado del cliente
