# Manual de Usuario - CRM UMYFA

## ¿Qué es esta aplicación?

Es un sistema simple para gestionar clientes y sus notas. Puedes agregar información de contacto y escribir notas sobre cada cliente.

## Cómo acceder

Abre tu navegador web y ve a:

https://app-umyfa-qxxzhw9oc-luisernestosalas-2775s-projects.vercel.app


## Funciones principales

### 1. Ver lista de clientes
- Al abrir la app verás todos tus clientes
- Cada cliente muestra: nombre, teléfono y email

### 2. Agregar nuevo cliente
1. Busca el botón **"Nuevo Cliente"** o **"+"**
2. Completa los campos:
   - Nombre (obligatorio)
   - Teléfono
   - Email
3. Haz clic en **"Guardar"**

### 3. Ver detalles de un cliente
- Haz clic sobre cualquier cliente de la lista
- Verás toda su información y sus notas

### 4. Editar cliente
1. Abre los detalles del cliente
2. Busca el botón **"Editar"**
3. Modifica la información
4. Guarda los cambios

### 5. Agregar notas
1. Abre los detalles
